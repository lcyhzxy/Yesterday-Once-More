API Endpoints

GET /api/health

Health check endpoint. Returns service status, available motion modes, styles, and configuration limits.

curl http://localhost:5000/api/health

POST /api/generate

Generate a dynamic video from an uploaded image. Supports multipart/form-data or JSON with image_url.

Parameters:
imageImage file (multipart/form-data)
motion_modeMotion effect (default: breathing)
styleVisual style (default: warm_memories)
durationVideo duration in seconds (default: 5, max: 30)
fpsFrames per second (default: 30)
resolutionWIDTHxHEIGHT (default: 1280x720)
intensityEffect intensity 0.0-1.0 (default: 0.5)
curl -X POST http://localhost:5000/api/generate \
  -F "image=@photo.jpg" \
  -F "motion_mode=breathing" \
  -F "style=warm_memories" \
  -F "duration=5"

GET /api/status/<task_id>

Check the status of a video generation task. Returns progress and result when completed.

curl http://localhost:5000/api/status/[task_id]

GET /api/video/<task_id>

Download the generated video file.

curl -O http://localhost:5000/api/video/[task_id]

GET /api/motions

Get list of available motion modes.

GET /api/styles

Get list of available video styles.

POST /api/preview

Generate a quick 3-second preview video for testing.

Motion Modes

breathing, swing, pan, zoom, shake, wave, pulse, float, rotation, ken_burns

Video Styles

original, warm_memories, cool_vintage, black_white, sepia, cinematic, dreamy, vivid, dramatic, soft_focus
